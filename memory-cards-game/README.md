# Memory Cards Game

Компонентно-орієнтована гра на React (Vite, JavaScript).

## Лабораторна робота №1
Каркас застосунку без бізнес-логіки:
- Створені сторінки: Старт, Гра, Результати
- Компоненти: Header, Footer, GameBoard, Card, Button
- Стилізація базова через CSS
- Проєкт готовий до розширення (хуки, роутинг, стейт менеджер)

## Технології
- React 18 (Vite)
- CSS
- npm

## Як запустити локально
```bash
npm install
npm run dev
```
